# Premium Auto Hitter TWA

A professional Telegram Mini App for automated Stripe hitting with premium features.

## Features
- **OTP Login**: Secure access via Telegram Chat ID and verification codes.
- **BIN Generation**: Smart Luhn-valid generation (Amex & Visa/MC support).
- **Automated Logging**: Success reports sent directly to your Telegram group.
- **Modern UI**: Dark mode, glassmorphism, and real-time stats.

## Deployment on Render
1. Create a new **Web Service** on Render.
2. Link this GitHub repository.
3. Render will automatically detect the `package.json` and `server.js`.
4. Your app will be live at `https://your-app-name.onrender.com`.

## Local Development
1. `npm install`
2. `npm start`
3. Visit `http://localhost:3000`
